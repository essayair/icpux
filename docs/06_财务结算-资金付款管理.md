# 矿产供应链运营管理系统
## 06 财务结算 · 资金与付款管理

---

## 角色定位

财务结算是**业务流转化为资金流的收口**，承接合同条款与收货数据，生成结算单、付款申请、发票，完成货款的收付款管理。同时管理运费归集、开票合规与资金占用计息，为管理层提供资金状态的完整视图。

> **典型岗位**：结算审核、税务合规、资金/物产对接（如黄蕾、王静、徐甜甜）

### 一、结算单制作

#### 1.1 结算单来源

结算单由系统根据以下数据自动计算生成，支持人工调整确认：

| 输入数据 | 来源 | 说明 |
|---------|------|------|
| 结算数量 | 磅单净重 - 扣水量 - 扣杂量 | 以哪个单据数量为准由合同约定（磅单/入库单/质检单） |
| 合同单价 | 采购/销售合同 | 约定单价；不定价合同需等待定价单确认 |
| 付款比例 | 合同结算条款 | 预付款比例（通常约85%）、尾款比例 |
| 扣重规则 | 合同+质检结果 | 扣水扣杂公式由合同约定，质检数据触发计算 |
| 付款账期 | 合同结算条款 | 约定账期天数，系统自动推算最晚付款日 |

#### 1.2 结算单计算逻辑

```
结算数量 = 磅单净重 − 扣水数量 − 扣杂数量
（扣重数量作"结算利息"单独处理，单独开票）

预付款金额 = 结算数量 × 合同单价 × 预付款比例（约85%，抹零处理）
尾款金额  = 结算数量 × 合同单价 − 预付款 − 其他扣款

最终货款 = 合同总金额 − 已付预付款 − 其他扣款
```

#### 1.3 审核要点

| 审核项 | 说明 |
|--------|------|
| 资料完整性审核 | 磅单、化验单、入库单等原始单据是否齐全 |
| 数量核对 | 结算数量与现场汇总数据是否一致 |
| 价格校验 | 结算单价是否与合同一致（或定价单已确认） |
| 税务合规 | 增值税税率、开票口径是否符合规定 |
| 扣重处理 | 扣重数量是否单独处理为"结算利息"并单独开票 |

> **操作原则**：台账统计仅以群内一手原始数据（磅单、化验单、入库单）录入台账，不参与审核。审核与录入职责分离。

### 二、付款审批

#### 2.1 付款申请流程

```
结算单审核通过
    → 生成付款申请
    → 按金额和部门配置的审批链
    → 审批通过
    → 执行付款（向供应商/资金方）
    → 银行回单归档
```

#### 2.2 关键操作节点

**向物产（资金方）提交审批**：
```
① 结算审核确认无误后，向王静下达指令：提交物产的资料清单及开票规则
② 王静向物产提交：结算单、质检单、入库单等审批资料
③ 追踪物产放款进度（"当日上午提交、下午到账"）
④ 物产放款到账后通知黄蕾
```

#### 2.3 付款方式

| 付款方式 | 说明 |
|---------|------|
| 电汇 | 常规银行转账 |
| 承兑汇票 | 商业承兑/银行承兑 |
| 信用证 | 适用于外贸场景 |

### 三、发票管理

#### 3.1 开票规则

**销项发票（开给客户）**：

| 规则 | 说明 |
|------|------|
| 正常货款发票 | 按结算数量×不含税单价开具增值税专用发票 |
| 扣重利息发票 | 扣重数量对应的金额单独开为"结算利息"，单独出具发票 |
| 税率规则 | 按合同约定税率（通常13%增值税）开具 |
| 开票时机 | 化验合格确认后才可开票 |

**进项发票（收自供应商）**：

| 管理事项 | 说明 |
|---------|------|
| 发票接收 | 供应商提供的增值税专用发票 |
| 合规查验 | 通过金税系统接口查验发票真伪 |
| 认证结果 | 记录增值税发票抵扣认证结果 |
| 作废管理 | 发票作废流程及替换单据管理 |

#### 3.2 发票流程

```
发票开具（黄蕾指导开票）
    → 发票传递下游客户/巨化方（王静负责）
    → 取得签收记录
    → 发票与结算单匹配归档
    → 财务入账
```

## 完整业务操作流程

**标准结算流程（预付款）**：

```
① 现场数据确认（郑学、郑小明 → 每日18:00前汇总）
② 现场负责人（陈飞祥）核对磅单、化验单 → @张妍确认"数据无误"
③ 陈飞祥 @黄蕾 通知启动结算流程
④ 黄蕾（结算审核）：
    → 审核资料完整性
    → 按约定比例（约85%抹零）计算预付款
    → 扣重数量作"结算利息"单独处理
⑤ 黄蕾 → 王静：下达物产审批指令，提供资料清单及开票规则
⑥ 王静向物产提交：结算单、质检单、入库单等审批资料
⑦ 追踪物产放款（当日上午提交、下午到账目标）
⑧ 物产放款 → 王静通知黄蕾到账金额
⑨ 黄蕾指导开票（扣重部分单独开"结算利息"，确保税务合规）
⑩ 王静：发票传递客户/巨化，取得签收记录
```

**最终结算流程（尾款）**：

```
① 巨化出具最终结算单（总数量、总金额、扣重汇总）
② 黄蕾据最终结算单计算尾款：
    总货款 − 已付预付款 − 其他扣款
③ 向物产提交最终结算资料，跟进尾款支付
④ 尾款到账 → 通知黄蕾、陈飞祥 → 完成全部资金支付
⑤ 出具全周期结算汇总表（三方共同确认）→ 合同关闭
```

## 开票合规要点

| 场景 | 开票处理 |
|------|---------|
| 正常货款 | 按实际结算数量×不含税单价开具增值税专用发票 |
| 扣重利息 | 扣重数量单独开为"结算利息"发票（确保税务合规） |
| 多期结算 | 每期分别开具对应发票 |
| 赊销场景 | 确认实际收款条件后再开票 |
