# 矿产供应链运营管理系统
## 05 仓储库管 · 库存管理

---

## 角色定位

仓储库管是**货物进入仓库后的管理中枢**，以批次为核心记录货物的数量、位置与质量状态，连接采购入库与销售出库两端，为财务成本核算和风控敞口管理提供库存数据基础。

> **典型岗位**：仓库管理员、台账录入内勤（如徐甜甜、海文岗）

---

## 业务链位置

```
【物流运营】（到货确认，物流入库单）
【现场一线】（质检合格，磅单净重）
          ↓
【仓储库管 · 库存管理】← 你在这里
     ↙              ↘
采销业务员           财务结算
（履约进度跟踪）    （批次成本核算）
                         ↓
                    风控/管理者
                  （库存敞口计算）
```

---

## 上游任务流：开始操作前需确认

| 前置条件 | 来源 | 说明 |
|---------|------|------|
| 物流入库单已确认 | 物流运营 | 货物到厂、过磅完成，物流运营确认物流入库单后通知库管 |
| 质检结论已出具 | 现场一线（质检化验） | 质检合格才可创建业务入库单，熔断货物不可入库 |
| 磅单净重已确认 | 现场一线（地磅） | 磅单净重是业务入库的数量依据，必须先于入库单生成 |
| 采购合同已生效 | 采销业务员 | 业务入库单关联对应采购合同，跟踪合同履约进度 |

## 核心功能模块

### 功能清单总览

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|--------|
| 业务入库单 | 引用物流入库单创建业务入库，完成收货确认，确认货权及库存形成 | P0 |
| 销售提单/出库单 | 按销售合同执行出库，扣减库存 | P0 |
| 过磅管理 | 磅单本地上传关联，采集双磅单，拍照留存 | P0 |

| 报损/报溢 | 处理库存盘点差异、损耗和溢余 | P1 |
| 批次成本核算 | 按批次加权实际成本计算库存成本 | P1 |

### 一、业务入库单

#### 1.1 功能说明

引用物流入库单创建业务入库，完成**收货确认，确认货权及库存正式形成**。财务确认并过账后，批次成本开始计算。

**数据来源**：物流入库单 + 采购合同 + 质检结果

#### 1.2 关键规则

| 规则 | 说明 |
|------|------|
| 质检前置 | 仅质检结论为"合格"的批次可创建业务入库单 |
| 磅单数量 | 入库数量以磅单净重为准，自动从磅单带入 |
| 合同关联 | 业务入库单关联采购合同，系统自动更新合同履约进度 |
| 预出库规则 | 业务入库单**未过账时**（业务状态"业务已确认"、入账状态"未生成"），**可以先做销售提单（预出库）** |
| 过账操作 | 财务"确认"和"过账"合并为一个按钮"确认并过账"，过账后库存正式计入账面 |

#### 1.3 特殊业务场景

| 场景 | 说明 |
|------|------|
| 无采购入库 | 支持创建新批次入库（非采购来源的库存形成） |
| 商品编码转化 | 入库时支持物料编码转化映射 |
| 定价调整单 | 价格确认后通过定价调整单更新批次成本 |

#### 1.4 入库关联链

```
物流入库单
    + 磅单净重（数量依据）
    + 质检报告（合格结论）
    + 采购合同（履约关联）
         ↓
    业务入库单（库存形成）
         ↓
    批次成本核算开始计算
```

### 二、过磅管理（仓库侧）

#### 2.1 功能说明

电脑本地磅单上传到系统，手动关联物流车辆，核对供货/收货单位。

| 功能 | 说明 |
|------|------|
| 磅单上传 | 从地磅电脑本地导出/上传磅单数据到线上系统 |
| 双磅单采集 | 同时录入供方磅单（供应商/矿山端）和我方现场磅单 |
| 磅单关联 | 手动关联物流运单和车辆，核对货物来源 |
| 拍照留存 | 上传磅单拍照，图片与磅单记录绑定存档 |

> **操作说明**：一期共用地磅设备但不共用本地数据，由仓库岗位同事在地磅电脑手动上传磅单数据到系统。

---

### 三、销售出库（销售提单）

#### 3.1 功能说明

销售提单/发货通知单是销售出库的核心单据，关联销售合同，向下触发物流出库和库存扣减。

#### 3.2 支持的出库模式

**常规出库**：
```
销售合同创建提单 → 拣配库存 → 物流出库 → 过磅确认 → 库存扣减
```

**直拨出库**：
```
供应商直接发货给客户（不经我方仓库）
→ 系统补完入库+出库流程（维持数据完整性）
```

**销售退货**：
```
支持退货到原批次（合并）或创建新批次入库
```

#### 3.3 关键规则

- 出库必须关联销售合同和库存批次
- 出库数量不可超过可用库存量
- 出库时依入库磅单生成出库磅单（必要时）

---

### 四、库存查询（业务库存）

| 查询维度 | 说明 |
|---------|------|
| 按物料 | 特定物料（如萤石粉精矿）当前总库存 |
| 按批次 | 每个批次的来源、入库数量、质检状态、已出库数量、剩余数量 |
| 按仓库/库位 | 每个库位当前存放的货物和数量 |
| 按部门 | 不同业务组归属的库存 |
| 库存日报 | 每日库存变动明细（入库/出库/移库/调整） |

---

### 五、库存调拨

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 仓库间调拨 | 货物从一个仓库移到另一个仓库 | P1 |
| 库位间调拨 | 同仓库内库位间移动 | P1 |
| 原批次调拨 | 保持原批次号，更新位置信息 | P1 |
| 新批次创建 | 调拨时新建批次（如分批存放场景） | P1 |
| 销售移库 | 内部利润中心间按移库核算（利润中心A → 利润中心B） | P1 |

> **销售移库说明**：当货物从一个利润中心转移到另一个利润中心时，系统按移库金额核算内部利润，而非等待最终销售完成。

---

### 六、报损 / 报溢

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 盘点录入 | 录入实际盘点数量，系统对比账面库存 | P1 |
| 差异单生成 | 账面 vs 实盘，生成盘点差异单 | P1 |
| 报损处理 | 差异为负（实际少于账面），生成报损单，关联审批流程，更新库存 | P1 |
| 报溢处理 | 差异为正（实际多于账面），生成报溢单，关联审批流程，更新库存 | P1 |

---

### 七、批次成本核算
**优先级：P1**

| 内容 | 说明 |
|------|------|
| 成本构成 | 货款（采购单价×入库数量）+ 运费分摊 + 其他费用分摊 |
| 分摊方式 | 货款/费用按重量比例分摊到批次 |
| 加权平均 | 同物料多批次按加权平均实际成本计算综合单位成本 |
| 出库成本 | 出库时按批次成本确认销售成本 |
| 适用场景 | 普通采购入库、委托加工入库、移库核算等多种场景 |

---

### 八、库存报表
**优先级：P1**

月底自动出具以下报表供财务使用：
- **库存台账**：按批次列示期初/本期入库/本期出库/期末数量和金额
- **收发存汇总**：按物料汇总当期收货量、发货量、期末库存量
- **库龄分析**：按批次列示入库日期和在库天数，识别滞销风险

## 完整操作流程

**采购入库标准流程**：

```
① 接收物流运营通知（物流入库单已确认）
② 核实质检结论（确认质检合格）
③ 在系统中创建业务入库单
    → 引用物流入库单
    → 磅单净重自动带入（入库数量）
    → 关联采购合同
    → 关联质检报告
④ 业务确认（库存批次生成，状态"业务已确认"）
⑤ 上传磅单照片（双磅单采集留存）
⑥ 财务确认并过账（库存正式形成，批次成本开始计算）
⑦ 通知采销业务员：收货完成，可发起结算
```

**销售出库标准流程**：

```
① 接收销售提单（由采销业务员下达）
② 确认库存可用数量（质检合格、未预占的库存）
③ 拣配库存批次
④ 通知物流运营安排出库运输
⑤ 出库过磅（地磅数据确认出库数量）
⑥ 在系统中创建物流出库单/业务出库单
⑦ 库存扣减（批次剩余数量更新）
⑧ 关联销售合同（销售合同履约进度更新）
```

## 下游任务流：你的操作触发

| 下游角色 | 触发事项 | 说明 |
|---------|---------|------|
| 采销业务员 | 业务入库单确认 | 采购合同履约进度更新，系统触发结算流程 |
| 财务结算 | 批次成本核算完成 | 批次成本数据为结算和报表提供基础数据 |
| 财务结算 | 库存报表月底出具 | 收发存汇总为财务月末账务处理提供依据 |
| 物流运营 | 销售提单确认出库 | 物流运营安排发货运输 |

## 模块关联关系

| 关联模块 | 关系说明 |
|---------|---------|
| 物流管理 | 到货/发货触发入库/出库单；物流入库单是业务入库单的前置 |
| 质检化验 | 入库批次触发质检；质检结论决定批次是否可用入库 |
| 地磅管理 | 过磅净重作为入库数量依据；磅单关联入库单 |
| 采销系统 | 出入库关联合同订单，跟踪合同履约进度 |
| 财务结算 | 批次成本数据为成本核算和资金计息提供基础 |
| 主数据管理 | 仓库、库位、物料取自主数据 |
